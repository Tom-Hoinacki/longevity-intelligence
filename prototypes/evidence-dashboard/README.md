# Evidence dashboard prototype

Standalone, dependency-free prototype for the first Longevity Intelligence evidence workspace. All content is clearly illustrative and not medical advice.

## Run

From this directory:

```powershell
python -m http.server 4173
```

Open http://localhost:4173. No install or build step is required.

See [DESIGN_HANDOFF.md](DESIGN_HANDOFF.md) for the design handoff.
